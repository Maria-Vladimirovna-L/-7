# # Expense Tracker (Трекер расходов)

## Автор
Большаков Кирилл Евгеньевич

## Описание
Графическое приложение для ведения учёта личных финансов. Позволяет добавлять расходы, указывать сумму, категорию и дату. Реализована фильтрация по категории и дате, а также подсчёт общей суммы расходов за выбранный период. Все данные сохраняются локально в файл `expenses.json`.

## Требования
- Python 3.8+
- Библиотека `tkinter` (встроена в стандартный дистрибутив Python)

## Установка и запуск
1. Клонируйте репозиторий:
   ```
   git clone https://github.com/yourname/expense-tracker.git
   ```
2. Перейдите в папку проекта.
3. Запустите приложение:
   ```
   python main.py
   ```

## Использование
1. Заполните поля: **Сумма**, **Категория**, **Дата** (в формате ГГГГ-ММ-ДД).
2. Нажмите кнопку **«Добавить расход»**.
3. Используйте фильтры **«Категория»** и **«Дата»** для просмотра истории.
4. Нажмите кнопку **«Подсчитать сумму»**, чтобы узнать общие траты за отфильтрованный период.
5. Данные сохраняются автоматически при добавлении и загружаются при запуске программы.
6. Для очистки фильтров нажмите кнопку **«Сбросить фильтр»**.

## Примеры использования (тесты)
*   **Добавление:** Сумма — `500`, Категория — `Еда`, Дата — `2026-04-28`.
*   **Фильтрация:** Введите `Еда` в поле «Категория» и нажмите «Фильтровать».
*   **Подсчёт:** После фильтрации по категории «Еда» нажмите «Подсчитать сумму», чтобы увидеть итог.
